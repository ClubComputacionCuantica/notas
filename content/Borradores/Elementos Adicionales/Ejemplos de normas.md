* La _[norma](../../Elementos%20de%20Álgebra%20Lineal/Producto%20interno%20y%20norma/norma.md) supremo_, también conocida como la _norma infinito_, se define para un vector $\mathbf{v} \in \mathbb{C}^n$ de la siguiente manera:$$
\| \mathbf{v} \|_{\infty} = \max \{ |v_i| : i = 1, 2, ..., n \}$$ donde $v_i$ son los componentes del vector $\mathbf{v}$.

* La _norma supremo_ para el [espacio vectorial de las funciones continuas acotadas](espacio%20vectorial%20de%20las%20funciones%20continuas%20acotadas) con valores en $\mathbb{C}$ en un [espacio topológico](Topología/Espacio%20topológico.md) $X$, denotado como $C_b(X, \mathbb{C})$, se define como:
$$
\| f \|_{\infty} = \sup \{ |f(x)| : x \in X \}.
$$
* La _norma 1_, también conocida como la _norma Manhattan_ o _norma taxicab_, se define para un vector $\mathbf{v} \in \mathbb{R}^n$ de la siguiente manera:$$
\| \mathbf{v} \|_{1} = \sum_{i=1}^{n} |v_i|$$donde $v_i$ son los componentes del vector $\mathbf{v}$.
