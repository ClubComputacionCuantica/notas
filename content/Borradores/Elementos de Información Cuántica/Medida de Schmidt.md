[^1][^2]Dado un [estado cuántico](../Elementos%20de%20Mecánica%20Cuántica/Postulado%201.%20El%20estado%20de%20un%20sistema.md) $\ket{\psi}\in \mathcal{H}^{(1)}\otimes\dots\otimes\mathcal{H}^{(n)}$ en un sistema con $n$ componentes, podemos escribir a $\ket{\psi}$ en la forma
$$
\ket{\psi}= \sum_{i=1}^R \xi_i\ket{\psi_i^{(1)}}\otimes\dots\otimes\ket{\psi_i^{(n)}}
$$
donde $\xi_i\in\mathbb{C}$ para cada $i=1,\dots,R$, y $\ket{\psi_i^{(k)}}\in\mathcal{H}^{(k)}$ para cada $k=1,\dots,n$.
Se define la _medida de Schmidt_ asociada con el estado $\ket{\psi}$ como
$$
E_S(\ket{\psi})=\log_2(r)
$$
donde $r$ es el mínimo $R$ sobre todas las descomposiciones anteriores.

It can be extended to the entire state space (and not only the extreme points) via a convex roof extension. ?
